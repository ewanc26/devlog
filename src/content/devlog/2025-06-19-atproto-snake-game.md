## atproto-snake

Browser-based Snake game with AT Protocol integration. Players log in via OAuth (DPoP, PAR), play, and submit scores as AT Protocol records using a custom `app.ewanc26.snake.score` lexicon. Features canvas rendering, touch controls, countdown, death animation, and score validation proofs. Later simplified auth with automatic server discovery.
