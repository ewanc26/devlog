### Git Migration
After nearly a year of informal development, I've officially migrated `ewancroft.uk` to a git-tracked Hugo repository. This move brings much-needed version control and automation to the project.

### Initial Setup
- **Hugo:** Chose Hugo for its speed and simplicity.
- **Content:** Started with a few initial posts and poems.
- **Identity:** Set up the basic identity links, including Fediverse and GitHub.
- **Automation:** Configured a GitHub Actions workflow for automatic deployment.
