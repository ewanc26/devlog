## @ewanc26/opal

A cross-platform Bluesky migration tool supporting Twitter, Mastodon, Threads, and Nostr as sources. Thread splitting for long posts, TID-based rkey generation via `@ewanc26/tid`, rate-limited publishing, and a web interface at opal.croft.click.
