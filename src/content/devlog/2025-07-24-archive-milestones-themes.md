### Milestone Recognition
I've added a milestone system to the blog archive. It automatically recognizes significant post counts and durations, adding a sense of progression to the historical timeline.

### Archive & Pagination
- **Pagination:** Added support for loading all blog posts with proper pagination, moving away from a single massive fetch.
- **Archive Header:** Rewrote the archive header using Tailwind CSS to be more responsive and consistent with the rest of the site.
- **Stats:** Consolidated date formatters and added word count analytics to the archive view.

### Werewolf Theme
Introduced the "Werewolf" theme, a new high-contrast aesthetic with both light and dark mode variants. This expansion is part of a broader effort to make the site more customizable.
