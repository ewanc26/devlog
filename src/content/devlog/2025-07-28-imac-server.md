## imac server attempt

Attempted to repurpose a late-2008 iMac as a home server. The machine had been sitting on the bedroom floor for two and a half years. Managed to brick it by resetting the OS without creating an installation USB first. Multiple failed attempts to create a bootable installer. Eventually gave up and got an M2 Mac Mini instead — which became both the daily driver and the actual server. The iMac remains on the floor, gathering dust, as a monument to ambition exceeding execution.
