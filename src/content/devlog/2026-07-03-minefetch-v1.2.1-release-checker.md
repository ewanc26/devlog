## minefetch

The Minefetch Paper plugin (which reports host system info in-game) shipped **v1.2.1** and gained self-update and refresh ergonomics.

### changes

- **v1.2.1**: bundled a Minefetch-specific fastfetch config and a sysinfo watcher.
- Cross-platform host support — Windows, Linux, and macOS.
- On-demand host sysinfo refresh driven by a trigger file.
- Added a GitHub release update checker.
- Initial commit of the Minefetch plugin itself.
