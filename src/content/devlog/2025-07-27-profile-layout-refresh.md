## profile redesign

Redesigned the website profile component. The old layout was centred — avatar floating above a sea of centred text, disconnected from the banner. Moved to a left-aligned layout where the avatar sits against the banner edge, with profile text flowing naturally below. Better visual hierarchy and connection between the banner and profile information.
