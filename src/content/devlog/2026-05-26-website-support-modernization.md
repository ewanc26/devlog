### Support Page Refactor
Redesigned the support page to better reflect the project's patronage model.
- **Patronage Components:** Added new components for highlighting various ways to support the project, including both financial and non-financial contributions.
- **FAQ:** Refactored the FAQ cards for better readability and a more consistent visual style.
- **CTA:** Added a non-financial Call-to-Action for users who want to contribute in other ways.

### Stability & Maintenance
- **Build Fixes:** Resolved several build and type errors that were causing CI/CD failures.
- **Modernization:** Modernized state management in several core components to align with the latest Svelte 5 patterns.
- **Version Bump:** Bumped the main website to v11.12.1.
