## nixos return

Put the Dell Inspiron 3501 back on NixOS after trying Bazzite. Bazzite is fine — atomic, immutable, works — but NixOS keeps pulling back. The declarative configuration, the ability to reproduce the entire system from a Git repository, the rollback support. The laptop isn't the daily driver, so the config is minimal: Firefox, VS Code, Spotify, Discord, Steam. Still learning as things go, but the config is in the flake and it works.
