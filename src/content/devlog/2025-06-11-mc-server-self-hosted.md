## minecraft self-hosting

Replaced Aternos free hosting with a self-hosted Minecraft server on the M2 Mac Mini. Aternos's 10-minute startup queues, sluggishness, and general unreliability became intolerable. The Mac Mini was already the daily driver — running a Minecraft server alongside it was straightforward. Full control over server properties, no queues, no reliance on a third-party platform. This later fed into the NixOS Minecraft server configuration.
