## linkat-directory

A Linkat-powered user directory at directory.croft.click. Displays user profiles with banners, descriptions, and metadata. Supports hiding the owner card, array shuffling, and prioritises display names. Later migrated to Svelte 5 with `$props()` and switched to Slingshot for identity resolution.
