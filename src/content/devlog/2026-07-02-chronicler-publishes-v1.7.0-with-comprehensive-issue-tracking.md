The Chronicler project publishes v1.7.0 and adds comprehensive issue tracking. The release brings 34 commits of improvements since the June 30 tag, including Paper plugin compatibility fixes, comprehensive activity tracking, and automatic restoration of the latest archived issue.

### releases
- v1.7.0 launched
- v1.6.0, v1.5.1, v1.5.0, v1.2.1, v1.2.0 released

### fixes
- Paper 26.2 compatibility: Replace BookMeta.pages() with addPages()
- Fix Paper plugin command registration, legacy color codes, messages.yml save
- Paper plugin startup crash fix with multi-provider LLM support

### features
- Add comprehensive activity tracking system
- Add automatic release updates with automatic book delivery
- Restore latest archived issue on startup
- Add editorial and publishing controls

### improvements
- Darken book text for better readability
- Allow all players to discover archived issues

### organization
- Structure issues as newspaper articles
- License management: delete LICENSE, add GNU AGPL v3 license
