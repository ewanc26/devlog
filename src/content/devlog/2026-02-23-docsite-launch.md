## docsite

Launched a dedicated documentation site at docs.ewancroft.uk. Built with SvelteKit and published to AT Protocol via Sequoia. Uses a unified/remark/rehype pipeline for markdown rendering with table of contents, code highlighting, and math support. All project documentation lives here — READMEs are minimal and link to the docsite.
