### Rebranding & UX
Social Sync (formerly `atproto-connect`) has reached v0.5.0. This release focuses on a cleaner identity and a more robust security model.

- **ModMenu Integration:** Redesigned the configuration screen specifically for ModMenu, removing the old bespoke GUI in favor of a native Minecraft feel.
- **OAuth Migration:** Replaced server-side logins with a modern AT Protocol OAuth flow (PKCE, DPoP, PAR), ensuring that player credentials never touch the game server.

### Privacy & Consent
- **Consent System:** Implemented a new 4-category consent system for syncing Minecraft data to the AT Protocol. Players now have granular control over what statistics, achievements, and session data are shared.
- **Session Security:** Client-side session storage is now encrypted using AES-256-GCM, and all sync actions are logged for auditability.
- **Automatic Sync:** Added support for automatic syncing of Minecraft advancements and real-time server status snapshots.
