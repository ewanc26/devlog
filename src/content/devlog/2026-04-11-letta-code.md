## letta code adoption

Started using Letta Code on 8 April 2026. Unlike ephemeral chat agents, it has persistent memory — remembers context across sessions, learns project conventions, and builds up knowledge over time. In four days: 80 commits across 15 repositories. The workflow is describe what's needed, it explores the codebase, writes the code, commits, and pushes. Still processing whether this is good or unsettling. The productivity is real; the question is whether the quality holds up across the projects it touches.
