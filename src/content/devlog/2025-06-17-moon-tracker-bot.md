## bluesky-moon-tracker

Posts daily moon phase updates to Bluesky. Built in TypeScript with a multi-source API (Skytime, Farmsense, local fallback) and optional Ollama LLM generation for natural language descriptions. Later rewritten in Rust using `atrium-rs`.
